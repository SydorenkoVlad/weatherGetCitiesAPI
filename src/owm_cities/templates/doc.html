Doc Page

Api is available on Page.

To download source file visit Page, download and unpack city.list.json.gz.

Available Params

  • mode - "0" for "start with" search, "1" for "contains"
  • q - query
  • keys - comma-separated keys - available: ("name", "id", "country", "coord"). Default - name,id
  • limit - return n values
  • offset - offset

Example

Simple request:

curl -X GET 'http://{host}:{port}/api' \ -d mode=0 \ -d q=kiev

Response:

{"errors": [], "result": [{"name": "Kievskij", "id": 6697077}, {"name": "Kieve", "id": 2890943}, {"name": "Kieve", "id": 6548066}, {"name": "Kiev", "id": 703448}, {"name": "Kievka", "id": 1522605}]}
